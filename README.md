# 🤖 Setup de Ferramentas de Coding com IA

Guia completo de instalação e configuração das ferramentas usadas neste projeto.

## Arquivos deste guia

| Arquivo | Conteúdo |
|---|---|
| `01-ollama.md` | Instalação e uso do Ollama (modelos locais) |
| `02-aider.md` | Instalação e uso do Aider |
| `03-modelos-nuvem.md` | Gemini e Groq gratuitos na nuvem |
| `04-atalhos.md` | Aliases e atalhos de terminal |

## Stack utilizada

- **Ollama** — rodar modelos de IA localmente
- **Aider** — agente de coding via terminal
- **Gemini 2.5 Flash/Pro** — modelo gratuito Google (nuvem)
- **Groq + Llama 3.3 70b** — modelo gratuito ultrarrápido (nuvem)

## Recomendação de uso

```
CPU fraca / sem GPU  →  use aider-gemini ou aider-groq
CPU boa / 16GB+ RAM  →  use aider-local (qwen2.5-coder:14b)
Privacidade total    →  use aider-local (100% offline)
```
